# MultiTool

MultiTool is a Windows app that combines several everyday utility features in one place.

## What MultiTool offers

- Auto clicker with custom timing, click type, repeat mode, and position control.
- Global hotkeys with safer capture behavior while editing mappings.
- Macro recording and playback for repeated workflows.
- Screenshot tools, including area capture, video recording, and latest media status (latest screenshot + latest saved video).
- Installer/cleanup tools for common apps.
- Extra Windows utilities (for example shortcut conflict scanning, hardware checks, telemetry controls, and maintenance helpers).

## Quick start (beginner friendly)

1. Make sure you are on Windows.
2. In the repository root, right-click `run-to-start.bat` and run as administrator.
3. If Windows shows a UAC/admin prompt, click Yes.
4. Wait for the script to finish building and setup.
5. MultiTool launches automatically when complete.

## Run again later

- Open `MultiTool.exe` in the repository root.
- Or run `run-to-start.bat` as admin again whenever you want to rebuild and launch in one step.
