# MultiTool

MultiTool is a Windows app that combines several everyday utility features in one place.

## What MultiTool offers

- Auto clicker with custom timing, click type, repeat mode, and position control.
- Global hotkeys to start or stop actions quickly.
- Macro recording and playback for repeated workflows.
- Screenshot tools, including area capture and save options.
- Installer/cleanup tools for common apps.
- Extra Windows utilities (for example shortcut conflict scanning, hardware checks, and maintenance helpers).

## Quick start (beginner friendly)

1. Make sure you are on Windows.
2. Install the .NET 8 SDK.
3. In the repository root, double-click `run-to-start.bat`.
4. If Windows shows a UAC/admin prompt, click Yes.
5. Wait for the script to finish building and setup.
6. MultiTool launches automatically when complete.

That script handles build + runtime setup + startup launcher setup for you.

## Run again later

- Open `MultiTool.exe` in the repository root.
- Or run `run-to-start.bat` again whenever you want to rebuild and launch in one step.
